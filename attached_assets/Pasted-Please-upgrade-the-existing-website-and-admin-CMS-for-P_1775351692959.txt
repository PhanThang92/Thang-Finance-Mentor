Please upgrade the existing website and admin CMS for “Phan Van Thang SWC” by adding a lead magnet / downloadable resources / gated content system.

IMPORTANT:
- Write implementation and code in English.
- Keep all visible UI text in Vietnamese.
- Build on top of the current website, admin CMS, Supabase/Postgres, lead forms, mini CRM, and newsletter/email system.
- Do NOT redesign the whole website.
- Do NOT build an overcomplicated membership platform.
- Focus on a practical gated-content system that helps collect qualified leads and deliver useful resources in a premium, trustworthy way.

GOAL:
Create a gated content / lead magnet system so the website can:
1. offer downloadable resources
2. collect lead information before access when appropriate
3. deliver the resource cleanly
4. connect downloads to the CRM
5. trigger email follow-up if desired
6. help the site owner understand which resources attract the most interest

This system should support:
- PDFs
- checklists
- guides
- mini ebooks
- templates
- curated resource downloads
- gated content pages
- optional “email to unlock” flows

The tone must remain:
- premium
- calm
- useful
- non-spammy
- trust-based

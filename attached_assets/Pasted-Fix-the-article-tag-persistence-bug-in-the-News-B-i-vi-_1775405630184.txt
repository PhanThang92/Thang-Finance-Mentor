- sometimes tags do not load correctly in admin
- sometimes the admin edit form shows missing tags even though the article had tags before
- if I open an article, edit it, and save/update, the related tags may disappear
- frontend then shows missing tags because the article-tag relation was overwritten incorrectly

This suggests a broken sync between:
- stored article-tag relations
- admin edit form state
- update payload
- frontend display
